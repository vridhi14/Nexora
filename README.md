## 📖 About

Nexora is a modern full-stack real-time chat application built with the MERN stack. It provides seamless one-to-one messaging with a beautiful UI, multiple themes, voice-to-text support, image sharing, authentication, and a smooth user experience.

Whether you're chatting with friends or exploring modern web technologies, Nexora demonstrates scalable full-stack development with secure authentication and responsive design.

---

## ✨ Features

- 🔐 Secure Authentication using **Clerk**
- 💬 Real-Time Messaging
- 🖼️ Image Sharing
- 🎙️ Voice-to-Text Messaging
- 🔇 Toggle Voice Input On/Off
- 🌙 Dark & ☀️ Light Mode
- 🎨 Multiple Beautiful Themes
- 🖼️ Custom Chat Backgrounds
- 📱 Fully Responsive Design
- ⚡ Instant Toast Notifications
- 😊 Clean & Modern UI
- 🔄 State Management using Zustand

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/vridhi14/Nexora.git

cd Nexora
```

### Install Dependencies

Frontend

```bash
cd frontend
npm install
```

Backend

```bash
cd backend
npm install
```

---

## ⚙️ Environment Variables

Create a `.env` file inside your backend (and frontend if required).

Example:

```env
PORT=

MONGODB_URI=

CLERK_SECRET_KEY=
VITE_CLERK_PUBLISHABLE_KEY=

IMAGEKIT_PUBLIC_KEY=
IMAGEKIT_PRIVATE_KEY=
IMAGEKIT_URL_ENDPOINT=
```

---

## ▶️ Run the Project

Backend

```bash
npm run dev
```

Frontend

```bash
npm run dev
```
